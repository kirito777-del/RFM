# 电商用户RFM分析
* 基于 20万 条订单数据 | 用户分层 | 行为分析 | 销售策略
---

## 一、项目背景与目标
### 项目背景
用户价值度细分是了解用户价值度的重要途径，根据RFM模型基于订单数据对用户价值度进行分层，剖析不同层级用户消费行为
并施行''分治''销售策略，便于后续精细化运营不同类型的用户群体，根据不同群体做定制化或差异性的营销与关怀
### 项目目标
* 基于R,F,M三个维度对用户进行价值度分层，划分核心用户群体
* 挖掘不同用户群体消费特征，根据不同价值度和消费习惯制定对应营销策略
* 输出可落地的分层运营策略，提高产品销量与回头率

### 二、RFM模型构建
#### 1. RFM指标定义
| 维度 | 含义 | 计算方式 |
| :--- | :--- | :--- |
| R (Recency) | 最近一次消费时间间隔 | 当前日期 - 最后消费日期 |
| F (Frequency) | 消费频率 | 用户累计消费次数 |
| M (Monetary) | 消费金额 | 用户累计消费总金额 |

#### 2. 评分标准（三分法）
| 得分 | R值（最近消费） | F值（消费频次） | M值（消费金额） |
| :--- | :--- | :--- | :--- |
| 3分（高） | 近期活跃（间隔短） | 高频消费 | 高消费金额 |
| 2分（中） | 中期未活跃 | 中频消费 | 中等消费金额 |
| 1分（低） | 长期未活跃 | 低频消费 | 低消费金额 |

*注：根据数据分布的四分位数或业务经验划定阈值*

#### 3. 用户价值分层矩阵（3×3×3 = 27类）
| 用户类型 | R值 | F值 | M值 | 特征描述 |
| :--- | :--- | :--- | :--- | :--- |
| **顶层核心用户** |
| 高价值活跃用户 | 高 | 高 | 高 | 核心VIP，近期活跃且贡献高 |
| **中层潜力用户** |
| 高频低消用户 | 高 | 高 | 中/低 | 活跃但消费力不足，需提升客单价 |
| 沉睡高价值用户 | 中/低 | 高 | 高 | 曾经的高价值用户，需唤醒 |
| 潜力发展用户 | 高 | 中/低 | 高 | 消费能力强但频次低，培养习惯 |
| **基层普通用户** |
| 新用户 | 高 | 低 | 低 | 刚完成首次购买 |
| 一般活跃用户 | 中 | 中 | 中 | 普通用户，各方面表现中等 |
| 边缘用户 | 中/低 | 低 | 低 | 价值低，可减少营销投入 |
| 流失风险用户 | 低 | 中/高 | 中/高 | 近期未消费的高价值用户 |

## 五、项目成果总结

### 一、分析结论
1.  **用户价值分布**
    -   高价值活跃用户（3-3-3）占比X%，贡献了Y%的销售额
    -   头部效应明显，需重点维护top用户
    -   中等价值用户占比最大，是转化主力人群

2.  **关键洞察**
    -   复购率与客单价呈正相关
    -   不同层级用户转化关键节点

3.  **机会点**
    -   高频低消用户客单价提升空间大
    -   潜力发展用户频次转化是关键
    -   中等价值用户向高价值迁移路径


### 二、项目价值
-   建立了科学的用户价值评估体系（3×3×3分层）
-   实现了用户精细化运营的数据支撑
-   形成了可复用的RFM分析模板
